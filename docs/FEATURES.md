# Gwent Beta Restoration — Features

What this private server lets you do in the Gwent 0.9.24.3.432 open beta, and what
isn't built yet. This is a functional overview - for how any of it works, the
code is right here in the repo.

## The goal

Restore Gwent's open beta to a playable state — both offline and online —
now that the official GOG servers are gone. You run a private server that stands
in for the old backend, so the game works the way it did at the time, including
real matches against other people.

## What you can do

- **Make an account** and sign in.
- **Build decks** from your collection.
- **Play against the AI** in practice matches and challenges.
- **Play online against other people** on the server through casual
  matchmaking — the main event.
- **Buy and open kegs** to expand your collection.
- **Craft cards** with scraps.
- **Transmute cards** (upgrade to premium/animated).
- **Mill cards** you don't want for scraps.
- **Level up** and collect your level-up rewards.
- **Friends**, friend requests and friend-match invites.

## Not yet implemented

- **Ranked play** — ranked matchmaking, leaderboards, and seasons.
- **Arena mode.**
- **Buying meteorite powder.**
- **Trinket unlocking** — all trinkets are available from the start rather than
  being unlocked.
